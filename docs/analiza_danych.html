1. Opis zbioru danych

Zbiór danych mtcars (Motor Trend Car Road Tests) zawiera informacje o 32 samochodach z 1974 roku. Dane pochodzą z pakietu datasets w R.

Zmienne w zbiorze: - mpg - spalanie (miles per gallon) - cyl - liczba cylindrów - disp - pojemność silnika - hp - moc (horsepower) - wt - waga (1000 lbs) - i inne…

# grupowanie po liczbie cylindrów i obliczanie statystyk
summary_stats <- mtcars %>%
  group_by(cyl) %>%  
  summarise(
    mean_mpg = mean(mpg, na.rm = TRUE),  
    sd_mpg = sd(mpg, na.rm = TRUE),      
    n = n()                              
  )
